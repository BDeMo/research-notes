# v1.8.0 settings (canonical, reproducible)

The exact configuration behind every v1.8.0 cell. The GCM driver is **`experiments/run_recipe.py`** (env-var configured),
launched per-config (1/GPU) by **`experiments/sweep.sh`**; both live in `/mnt/persist/gcm` on each pod. Every run logs to
W&B (`uva-vast/gcm`) and writes `out/<base>/<TAG>.json` + `<TAG>_adapters.pt`.

## Phase routing (which loss trains which params) — `train.py`
Losses are split into two buckets with **disjoint** parameter updates (verified by per-loss `.grad` inspection):

| phase | losses | updates | mechanism |
|---|---|---|---|
| **Phase 1 — compression (train encoder)** | `recon`, `dev`, `kl`, `adv`, `contrast` | mem, proj, enc-LoRA, decoder, rec_slot | main encode runs with read-LoRA **off** (kl/dev never touch it); recon/adv/contrast **freeze read-LoRA** during their backward |
| **Phase 2 — gated generation (learn to read M)** | `task`, `distill` | read-LoRA only | student logits built on **`M.detach()`** → answer-CE never flows to the encoder |

**Consequence:** the answer CE no longer trains the encoder. The encoder is shaped *only* by Phase-1 losses, so a config must include ≥1 of {recon, dev, adv, contrast} or M stays at init (proj of frozen-base features). The reader (read-LoRA) is shaped only by task/distill against a fixed M.

**Contrastive (`L_contrast`, InfoNCE):** pull M-induced layer-`adv_layer` hidden `hM` toward its own full-ctx-induced `hF` (positive), push from a rolling 256-vector bank of other `hF` (negatives); temperature `GCM_CONTRAST_TEMP`. Encoder-only (read-LoRA frozen). At bs=1 the bank supplies negatives across steps.

## Anchor launch (toolace, one config) — example
```bash
CUDA_VISIBLE_DEVICES=0 PYTORCH_CUDA_ALLOC_CONF=expandable_segments:True \
GCM_TRAIN=toolace GCM_EVAL=toolace GCM_NTRAIN=6509 GCM_NVAL=256 \
GCM_K=16 GCM_DEPTH=half GCM_NORM=hard GCM_DISTILL=0.5 \
GCM_BATCH=1 GCM_ACCUM=8 GCM_MAX_STEPS=10000 GCM_PATIENCE=10 \
GCM_MAXCTX=4096 GCM_GOLD_MAX=32 GCM_GEN_MAX=128 GCM_GEN_BS=8 \
GCM_TAG=sw_toolace_anchor \
python -u experiments/run_recipe.py /mnt/persist/checkpoints/Qwen3.5-9B
```

knobs exported first; one config per GPU, all inside tmux.
