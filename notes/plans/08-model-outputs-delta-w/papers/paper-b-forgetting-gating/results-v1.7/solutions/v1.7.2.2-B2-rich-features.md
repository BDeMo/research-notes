# B2 · v1.7.2.2 — 富特征门（多层 hidden + attention 池化）

## 动机
现在判别器只看 base **单层 ℓ=18**、对 query 位置 **均值池化**——信息有限。B2 给门更强的输入：**多层** hidden + **attention 池化**（可学的 query 向量去 attend M-run 的 hidden），再融合 intrinsic 信号。建立在 B1（监督训练 + CV）之上。

## 详细做法
1. `signals` 扩展：记录 base 在 `[Mq;q]` run 下**多层**（如 ℓ∈{6,12,18,24,30}）query 位置 hidden 的池化向量（均值 + 末 token + 可学 attn-pool）。
2. 同时记 `[ctx;q]`(full) 对应层的同样池化，作为对照特征（Δ = M-run − full-run，可选，部署时 full 不可得故仅分析）。
3. 门 = 小 MLP / 浅 transformer，输入 = 多层池化向量 ⊕ intrinsic 信号；监督标签同 B1，CV。
4. 消融：层子集、池化方式（mean vs attn vs last）、是否加 intrinsic。

## 实现位置
- `svc/method.py::signals`：`output_hidden_states=True`，对选定层做 mean/last/attn-pool，写进 `rec["signals"]["hL{ℓ}_*"]`（注意体积：只存池化后的小向量或其投影）。
- `svc/disc_gate.py`：`--fit mlp` 支持高维特征 + CV。
- 跑：重跑 GCM（带富信号）`q35_B2_squad/hotpot`，再离线拟合。
