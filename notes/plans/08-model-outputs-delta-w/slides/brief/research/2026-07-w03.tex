\begin{frame}{Research question and method}
  \begin{block}{Question}
    When can an agent answer from a short learned state, and when should it return to raw evidence?
  \end{block}
  \begin{enumerate}\small
    \item Recurrently encode context chunks into soft memory while carrying earlier memory forward.
    \item Read memory through a small LoRA adapter on the same frozen language model.
    \item Keep an adapter-free bounded raw path on the same backbone.
    \item Use held-out confidence to study routing between the two paths.
  \end{enumerate}
  \begin{block}{Positioning}
    \footnotesize Compression ratio alone is not the contribution. Paper A studies the complete
    \textbf{compact-memory versus raw-evidence interface}, including paired failure, coverage, and cost.
  \end{block}
\end{frame}
