\begin{frame}{Recall: Why We Are Doing This}
  \begin{block}{The problem}
    Feeding an LLM a long context every turn is slow and expensive. We study whether we can \textbf{compress the
    context into a few latent ``memory'' tokens} a \emph{frozen} base model reads --- and, when that compression
    would lose what the query needs, \textbf{fall back to the full context} (``do-no-harm'').
  \end{block}
  \begin{itemize}
    \item \textbf{Last week:} a clean (leak-free, stable-training) study reframed the work from ``build a better
      compressor'' to a \textbf{capacity-bound diagnosis} + the \textbf{do-no-harm gate} it prescribes.
    \item \textbf{This week:} the supporting experiments actually \emph{landed} --- cross-model breadth, a module
      ablation, a \emph{deployable} gate threshold, and an honest read of what is still missing.
  \end{itemize}
\end{frame}

\begin{frame}{Preliminaries: Terms Used Today}
  \small
  \begin{itemize}
    \item \textbf{Frozen base}: the pretrained LLM; weights never change (so behavior is recoverable).
    \item \textbf{Learned memory / soft prompt}: $K$ trained latent vectors ($\approx$ ``compressed context'') the base reads.
    \item \textbf{Compressor}: the module that maps long context $\to$ memory (ours = GCM; rivals = ICAE/Beacon/500x/\dots).
    \item \textbf{Gate}: a per-input decision --- \emph{keep} the compressed memory (cheap) or \emph{fall back} to full context (safe).
    \item \textbf{Do-no-harm}: the gate never lowers quality below full context (by construction the base is recoverable).
    \item \textbf{AUROC}: how well a gate \emph{signal} ranks safe-vs-unsafe inputs (0.5 = chance, 1.0 = perfect).
    \item \textbf{recall@P95 / coverage}: how much we can safely compress at $\ge$95\% precision (the savings we can claim).
    \item \textbf{ctx$>$K}: items where the context is longer than the memory budget --- the only items where ``compression'' is real.
    \item \textbf{test-agnostic threshold}: a gate cutoff set \emph{without} test labels (so it is deployable on any dataset).
  \end{itemize}
\end{frame}

\begin{frame}{Verified: Module Ablation --- Only Distillation + Reconstruction Matter}
  \small
  \begin{columns}[T]
    \begin{column}{0.52\textwidth}
      \begin{block}{From the full method, remove one module}
        \scriptsize (Qwen3-8B; $\Delta$compress-acc vs full config; 2 benches)
        \begin{itemize}\scriptsize
          \item $-$ \textbf{distillation}: $-0.51$ / $-0.22$ \;\textbf{(collapse)}
          \item $-$ \textbf{reconstruction}: gate AUROC $0.75\!\to\!0.48$ \;\textbf{(gate dies)}
          \item $-$ task loss: bench-dependent (keep it)
          \item depth 16$\to$4, $K$64$\to$4, random init, $-$dev: $\approx$ neutral
        \end{itemize}
      \end{block}
    \end{column}
    \begin{column}{0.48\textwidth}
      \begin{block}{Takeaway}
        \scriptsize \textbf{Minimal sufficient method} = a \emph{tiny} encoder ($N\!\approx\!4$, $K\!\approx\!16$) + distillation +
        reconstruction (which powers the gate) + task loss. Everything else (big encoder, large $K$, alignment
        losses, OOD fixes) is droppable $\Rightarrow$ supports ``minimize-scale; the gate is the contribution.''
      \end{block}
    \end{column}
  \end{columns}
\end{frame}

\begin{frame}{Verified: A Deployable (Test-Agnostic) Gate Threshold}
  \small
  \begin{block}{The deployment question}
    In production you have no labels --- so you cannot tune the gate cutoff $\tau$ on the test set. \emph{Is there a $\tau$ rule that works on any dataset?}
  \end{block}
  \begin{itemize}
    \item \textbf{Yes, with a calibrated-probability signal} (base ``margin'' = top1$-$top2 prob): a fixed threshold is dataset-independent.
    \item \textbf{margin $\ge 0.95$ $\Rightarrow$ do-no-harm on all 20 model$\times$bench cells} (worst $-0.01$); coverage \emph{auto-adapts} (compress where safe, fall back where not).
    \item Intrinsic signals (reconstruction-based) are \emph{not} test-agnostic (arbitrary scale $\Rightarrow$ need per-dataset calibration).
    \item Signal ranking (T10): agnostic \texttt{neg\_entropy}/\texttt{conf} $\approx$ best intrinsic ($\sim$0.6 mean AUROC); others at chance.
  \end{itemize}
\end{frame}
