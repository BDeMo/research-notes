# Janus — broad fact-finding on the 7–9B cohort (2026-06-04)

> Goal (per user): pre-experiments = **find facts**. Define metrics broadly (information theory · representation geometry · parameter spectra · attention structure · forgetting-coupling) and look for **cross-family correlations** at a controlled scale. Scale axis deferred.

## Setup
- **Cohort (≈8B, native bf16, NO quantization; metrics computed in fp32)**: **Qwen3.5-9B** (hybrid 3:1 Gated-DeltaNet/full-attn, multimodal backbone), **GLM-4-9B** (Zhipu, cross-vendor), **Qwen3-8B**, **Qwen2.5-7B-Instruct**. A fact that holds in all four is architecture-robust, not a Qwen quirk.
- **Env**: all four run in one isolated venv (torch 2.11 / transformers 5.10) on ray so version is not a confound. Qwen3.5 required torch ≥2.7 (`float8_e8m0fnu`).
- **~30 per-layer metrics** across 5 angles + per-head coupling cross-linked (mean-pooled to layers). Code: `runs/janus_run.py` (`facts`, `intrinsic`), `runs/analyze_cohort.py`. Figures: `figs/12_facts_corr_*`, `figs/13_facts_traj_*`, `figs/14_facts_consistent.png`.

## Metric angles
1. **Representation geometry**: effective rank, repr-entropy, anisotropy, intrinsic-dim (TwoNN), adjacent-layer CKA, activation kurtosis/sparsity, residual-norm, update-norm, trajectory curvature.
2. **Information theory** (logit-lens + **tuned-lens-lite**): prediction entropy, KL-to-final, top-1 prediction depth.
3. **Parameter spectra** (per weight matrix, SVD): stable rank, effective rank, spectral norm, **HT-SR power-law α**, MLP gain.
4. **Attention structure** (per head): sink, retrieval, induction, attn-entropy/distance, KV/V norm, per-head spectral α.
5. **Forgetting coupling**: a-priori Fisher, gradient magnitude, **gradient-noise scale**; post-SFT weight drift & activation drift.

## The Janus-relevant fact (forgetting coupling, cross-family)
- **`dW_drift ↔ retrieval` (per head, mean-pooled) = +0.52** across all 4 families → **retrieval heads are where SFT weight-drift concentrates**, replicated on Qwen3.5 / GLM-4 / Qwen3 / Qwen2.5. This is the per-head, cross-family version of the H2 coupling: the read-side long-context site is the write-side drift site.
- Caveat: the earlier *single-model* "Fisher ↔ logit-lens-KL = +0.92" did **not** replicate as a clean cross-family per-layer fact (hd_fisher's only strong consistent partner is hd_grad_mag, a tautology). So the **drift↔retrieval** link is the robust forgetting fact; the Fisher↔info-layer link is model-specific.

## The "interesting" headline
A **single representation-collapse axis** runs through every family and ties three angles together:
**late layers** → residual norm ↑, prediction crystallizes (KL-to-final ↓, top-1 depth ↑), representations collapse (eff-rank ↓, anisotropy ↑), **massive activations emerge** (kurtosis ↑, sparsity ↑), and the **MLP down-proj spectrum** (stable rank) co-moves. Information formation, representation geometry, and parameter spectra are **the same phenomenon viewed three ways** — and it is vendor-independent (holds on GLM-4 as on Qwen).

## Caveats
- **Qwen3.5 hybrid**: only 8/32 layers are full-attention, so its per-head attention metrics (sink/retrieval/…) cover those 8 layers only (correctly remapped); per-layer representation/info/param facts use all 32 and are fully comparable.
- `fla`/`causal-conv1d` kernels not installed → Qwen3.5 uses the slower torch path (results unaffected).
- 4–8 generic text probes per model; per-layer ρ over 28–64 layers. Directional, not yet seed-banded.
